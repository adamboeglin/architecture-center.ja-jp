1. <span data-ttu-id="d41b2-138">ネットワークを接続する。ガバナンスによってサポートされるネットワーキングと IT セキュリティによって実行されます。</span><span class="sxs-lookup"><span data-stu-id="d41b2-138">Connect the networks - Executed by Networking and IT Security, supported by governance</span></span>
