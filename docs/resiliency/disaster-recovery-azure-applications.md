<span data-ttu-id="7f67c-558">シミュレーション テストでは、作業現場で小規模な現実的状況を作成し、チーム メンバーがどのように反応するかを観察します。</span><span class="sxs-lookup"><span data-stu-id="7f67c-558">Simulation testing involves creating small real-life situations on the work floor to observe how the team members react.</span></span> <span data-ttu-id="7f67c-559">また、シミュレーションでは、復旧計画で示されているソリューションの有効性もわかります。</span><span class="sxs-lookup"><span data-stu-id="7f67c-559">Simulations also show how effective the solutions are in the recovery plan.</span></span> <span data-ttu-id="7f67c-560">作成されたシナリオが実際のビジネスを中断することなく、それでいて実際の状況に近いものになるように、シミュレーションを実行します。</span><span class="sxs-lookup"><span data-stu-id="7f67c-560">Execute simulations so that the created scenarios don't disrupt actual business, while still feeling like real situations.</span></span>
